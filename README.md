Graph

    - Rows
    - Cell
        - Color
        - Symbol
        - Locked

Actions:
    - Select
    - Transform
    - Rotate
    - Clear
    - Fill
    - Copy
    - Cut
    - Paste
    - Outline
    - Undo
    - Redo
    - Set row amount
    - Set column amount
